# MFChanger

FC온라인 선수 미페(미니 페이스) 및 팀 크레스트를 쉽게 교체할 수 있는 프로그램입니다.

---

## 주요 기능

- **선수 미페 교체** — 원하는 선수를 검색해 이미지를 교체
- **공식 미페로 복원** — CDN에서 공식 이미지를 다운로드해 원상복구
- **팀 크레스트 교체** — ID를 입력해 팀 엠블럼 이미지를 교체
- **감독 얼굴 교체** — 감독 이름/팀명으로 검색 후 원하는 얼굴로 교체
- **변경 이력 및 복원** — 교체 전 원본을 자동 백업, 언제든지 되돌리기 가능
- **자동 업데이트** — 새 버전 출시 시 프로그램 내에서 바로 업데이트

---

## 사용 방법

### 선수 미페 교체

1. **미페 변경** 탭으로 이동합니다.
2. 검색창에 선수 이름을 입력하고 **검색** 버튼을 누릅니다.
   - 필터 버튼으로 시즌(TOTY, TOTS, UCL 등)을 좁혀서 찾을 수 있습니다.
3. 검색 결과는 **동명이인·같은 선수가 가나다순으로 그룹화**되어 표시됩니다.
   - 그룹 헤더(이름 행)를 클릭하면 해당 그룹을 접거나 펼칠 수 있습니다.
   - 검색 결과가 단 하나의 그룹이면 자동으로 펼쳐집니다.
4. 목록에서 원하는 선수 카드를 클릭하면 오른쪽에 현재 적용 중인 미페가 표시됩니다.
   - **현재 미페 / 공식 미페** 토글로 현재 적용 상태와 서버 원본을 비교할 수 있습니다. 공식 미페는 인터넷 연결이 필요합니다.
   - 미페 교체 후 목록 썸네일과 미리보기가 즉시 갱신됩니다.
5. **이미지 파일 선택** 버튼으로 교체할 이미지를 고르거나, 이미지를 드래그 앤 드롭합니다.
   - JPG, PNG, BMP, WEBP를 지원합니다.
   - 비정방형 이미지는 자동으로 중앙 크롭되어 적용됩니다.
   - **파일명이 `p{숫자}` 형식이면 해당 선수를 자동으로 검색합니다.** 자세한 내용은 아래 참고.
6. **미페 교체 실행** 버튼을 클릭하면 즉시 적용됩니다.

> **파일 위치 열기** 버튼을 누르면 해당 선수 이미지가 저장된 게임 폴더를 탐색기에서 바로 열 수 있습니다.

#### 미페 파일 자동 인식 (PID / SPID)

게임 미페 파일의 파일명은 `p{SPID}.png` 형식입니다.

- **SPID** — 9자리 숫자. 앞 3자리는 시즌 번호, 나머지 6자리가 선수 고유 번호(**PID**)입니다.
  - 예: `p271123456.png` → 시즌 271, PID 123456
- **PID 파일** — `p{PID}_25.png`처럼 파일명 뒤에 `_24`, `_25` 등 연도가 붙은 경우도 있습니다.

이미지 파일을 선택하거나 드래그 앤 드롭하면 파일명을 분석하여 다음과 같이 동작합니다.

| 파일명 예시 | 동작 |
|---|---|
| `p271123456.png` | SPID 9자리 인식 → PID `123456` 검색 후 해당 선수 카드 자동 선택 |
| `p123456.png` | PID 6자리 인식 → PID `123456` 검색 (자동 선택 없음) |
| `p123456_25.png` | PID 6자리 인식 (언더스코어 전까지) → PID `123456` 검색 |
| `example.png` | 일반 파일명 → 자동 검색 없음 |

### 공식 미페로 복원

프로그램 사용 이전에 이미 미페를 바꿔둔 경우 등, 백업 없이도 공식 이미지로 되돌릴 수 있습니다.

1. **미페 변경** 탭에서 복원할 선수를 선택합니다.
2. **공식 미페로 복원** 버튼을 클릭합니다.
3. CDN에서 공식 미페를 다운로드해 로컬 파일을 덮어씁니다.
   - 백업 설정이 켜져 있으면 기존 파일이 자동 백업되므로, 이후 **변경 이력**에서 다시 되돌릴 수 있습니다.
   - 인터넷 연결이 필요합니다.

---

### 팀 크레스트 교체

1. **크레스트 변경** 탭으로 이동합니다.
2. **ID 찾기** 버튼을 눌러 FC온라인 팀컬러 검색 페이지에서 교체할 팀의 크레스트 ID를 확인합니다. (확인 방법은 아래 참고)
3. 확인한 ID를 입력란에 입력하고 **미리보기** 버튼을 누르면 현재 적용된 크레스트를 확인할 수 있습니다.
4. **이미지 파일 선택** 버튼으로 교체할 이미지를 고르거나 드래그 앤 드롭합니다.
   - `[id].png`, `d[id].png`, `l[id].png` 형식의 파일명이면 ID가 자동으로 입력됩니다.
     예) `1234.png` / `d1234.png` / `l1234.png`
   - 입력란에 `d1234`, `l1234.png` 형식으로 직접 붙여넣어도 숫자 ID만 자동으로 인식합니다.
   - 권장 이미지 크기는 **256 × 256 픽셀 정방형**입니다. 비정방형 이미지는 자동으로 중앙 크롭됩니다.
5. **크레스트 교체 실행** 버튼을 클릭합니다.

#### 크레스트 ID 확인 방법

**팀컬러가 있는 경우**

**Step 1.** **ID 찾기** 버튼을 눌러 FC온라인 팀컬러 검색 페이지로 이동합니다.

**Step 2.** 원하는 팀을 검색하면 팀 엠블럼 이미지가 표시됩니다.

**Step 3.** 이미지를 **우클릭 → 다른 이름으로 저장**하면 파일명에서 ID를 확인할 수 있습니다. 파일명을 그대로 복사해 입력란에 붙여넣으면 자동으로 인식됩니다. `.png` 확장자가 포함된 상태로 붙여넣어도 됩니다.

---

**팀컬러가 없는 경우**

팀컬러 검색으로 찾을 수 없는 팀의 경우 아래 방법으로 ID를 확인합니다.

**Step 1.** 인게임에서 원하는 크레스트를 팀에 직접 설정합니다.

**Step 2.** FC온라인 홈페이지에 변경 사항이 반영될 때까지 잠시 기다립니다. 이후 홈페이지 내 자신의 프로필에 표시되는 크레스트 아이콘을 찾습니다.

**Step 3.** 크레스트 이미지를 우클릭합니다.

**Step 4.** **다른 이름으로 저장**을 클릭합니다.

<img src="docs/image/1.png" width="600">

**Step 5.** 저장 대화상자에서 파일명을 따로 수정하지 말고 바로 **Ctrl+C**로 복사합니다.

<img src="docs/image/2.png" width="600">

**Step 6.** MFChanger의 크레스트 ID 입력란에 **Ctrl+V**로 붙여넣고, **미리보기** 버튼을 클릭해 크레스트를 확인합니다.

<img src="docs/image/3.png" width="600">

**Step 7.** 설정한 크레스트가 잘 출력됩니다. 이제 바꾸려는 크레스트를 올린 뒤 변경해주시면 됩니다.

### 감독 얼굴 교체

1. **감독 변경** 탭으로 이동합니다.
2. 검색창에 감독 이름 또는 소속 팀명을 입력하고 **검색** 버튼을 누릅니다.
3. 검색 결과에서 원하는 감독을 선택하면 오른쪽에 현재 적용 중인 얼굴이 표시됩니다.
   - **현재 얼굴 / 공식 얼굴** 토글로 현재 적용 상태와 서버 원본을 비교할 수 있습니다.
   - 게임이 해당 감독의 얼굴을 아직 로컬에 캐시하지 않은 경우 현재 얼굴은 기본 이미지로 표시됩니다. 해당 감독이 포함된 팀을 인게임에서 만나거나 **공식 얼굴로 복원**을 실행하면 로컬 파일이 생성됩니다.
4. **이미지 파일 선택** 버튼으로 교체할 이미지를 고릅니다.
5. **감독 얼굴 교체 실행** 버튼을 클릭합니다.

> **공식 얼굴로 복원** 버튼을 누르면 CDN에서 공식 이미지를 다운로드해 로컬에 저장합니다. 로컬 파일이 없던 감독도 이 버튼으로 파일을 생성할 수 있습니다. 인터넷 연결이 필요합니다.

> 감독 변경 이력은 **변경 이력** 탭의 "감독 변경 이력" 섹션에서 확인할 수 있습니다.

### 변경 이력 및 복원

**변경 이력** 탭에서 지금까지 교체한 미페, 크레스트, 감독 얼굴 목록을 확인할 수 있습니다.

#### 이력 구조

**미페 / 크레스트**
- 같은 선수/크레스트를 여러 번 변경하면 **변경 횟수만큼 이력이 스택으로 누적**됩니다.
- 선수별로 그룹 카드 형태로 표시되며, 그룹 헤더를 클릭해 접거나 펼칠 수 있습니다.
- 그룹 내 항목은 **최신순**으로 표시됩니다. 가장 위 항목(최신)만 활성화되어 있고, 이전 항목들은 흐리게 표시됩니다.
- 각 항목에는 **변경 전 → 변경 후** 이미지가 나란히 표시됩니다.

**감독**
- 같은 감독을 여러 번 변경해도 이력은 **1건만 유지**됩니다 (가장 최근 변경으로 덮어씀).
- 백업 파일 없이 CDN으로만 복원되므로 중간 시점으로의 되돌리기는 지원하지 않습니다.
- 복원 버튼은 "공식 얼굴로 복원"으로 표시됩니다.

#### 복원

- **복원** 버튼은 가장 최신 항목에만 표시됩니다. 클릭하면 그 시점의 백업 파일로 게임 파일을 되돌리고, 해당 이력 항목이 제거됩니다.
- 이전 항목들을 순서대로 되돌리려면 복원을 반복하면 됩니다.
  - 예: A→B→C 순서로 변경했다면, 복원을 누를 때마다 C→B→A 순서로 되돌아갑니다.
- 백업이 없는 경우 교체 파일을 삭제하며, 게임 재실행 시 자동으로 원본으로 복구됩니다.

> **복원 후에도 이전 이력이 남아 있으면 계속 복원할 수 있습니다.** 예를 들어 커스텀 미페로 바꾼 뒤 공식 미페로 복원했더라도, 이력이 남아 있다면 공식 미페 복원 내역을 다시 복원해 커스텀 미페로 돌아갈 수 있습니다.

#### 삭제 (이력 정리)

삭제는 복원과 다릅니다. **게임 파일은 건드리지 않고** 백업 파일과 이력 기록만 제거합니다.

- 각 항목 왼쪽의 **체크박스**로 삭제할 항목을 선택한 뒤, 그룹 헤더의 **선택 삭제(N)** 버튼을 클릭합니다.
- 최신 항목을 삭제해도 현재 적용 중인 게임 파일은 바뀌지 않습니다. 이력 기록만 사라집니다.
- 중간 이력 항목을 삭제하면 해당 시점의 백업만 사라집니다. 나머지 항목의 복원 기능은 영향을 받지 않습니다.
- **전체 삭제** 버튼으로 해당 선수의 모든 이력과 백업을 한 번에 제거할 수 있습니다.

> **삭제와 복원의 차이 요약**
>
> | | 게임 파일 | 백업 파일 | 이력 기록 |
> |---|---|---|---|
> | **복원** | 백업으로 덮어씀 | 삭제됨 | 해당 항목 제거 |
> | **삭제** | 변경 없음 | 삭제됨 | 해당 항목 제거 |

#### 기타

- **폴더** 버튼으로 해당 항목의 백업 폴더를 탐색기에서 열 수 있습니다.
- 이력에서 복원해도 검색 목록 썸네일과 미리보기가 즉시 갱신됩니다.

### 설정

**설정** 탭에서 다음 항목을 관리합니다.

| 항목 | 설명 |
|------|------|
| FC온라인 설치 경로 | 게임이 설치된 폴더 경로. 기본값은 `C:\Nexon\EA SPORTS(TM) FC ONLINE` |
| 백업 설정 | 교체 전 원본 파일을 자동 백업할지 여부와 저장 위치 |
| 폰트 크기 | UI 텍스트 크기를 작게 / 보통 / 크게 / 매우 크게 중에서 선택. 재시작 없이 즉시 적용됩니다. |
| 선수 데이터 동기화 | 선수 목록과 시즌 정보를 최신 버전으로 업데이트 |
| 업데이트 | 시작 시 자동 업데이트 확인 여부 및 수동 확인 |

---

## 주의사항

- 변경 사항은 **게임 실행 중에도 실시간으로 적용**됩니다. 교체 후 다른 화면으로 이동했다가 돌아오면 바로 반영됩니다.
- FC온라인 업데이트 후 게임이 캐시를 초기화하면 교체한 이미지가 원래대로 돌아갈 수 있습니다.
- **백업 기능을 활성화해두면 이력을 통해 언제든지 원하는 시점으로 복원할 수 있습니다.** 백업이 꺼져 있으면 복원이 불가능하므로 사용을 권장합니다.

### 오프라인 환경 제한

인터넷 연결 없이도 기본 기능은 사용할 수 있지만, 일부 기능은 동작하지 않습니다.

| 기능 | 오프라인 |
|------|----------|
| 미페 교체 / 크레스트 교체 | 정상 동작 |
| 감독 얼굴 교체 | 정상 동작 |
| 현재 미페 / 현재 얼굴 프리뷰 | 정상 동작 |
| 복원 / 변경 이력 | 정상 동작 |
| 검색 목록 썸네일 | 정상 동작 (로컬 파일 기준) |
| 공식 미페 프리뷰 | 표시 안 됨 |
| 공식 얼굴 프리뷰 (감독) | 표시 안 됨 |
| 공식 미페로 복원 | 불가 |
| 공식 얼굴로 복원 (감독) | 불가 |
| 선수 데이터 동기화 | 불가 |
| 자동 업데이트 확인 | 불가 |
| 첫 실행 시 선수/감독 목록 로딩 | `.cache` 폴더 동봉 시 정상, 없으면 불가 |
